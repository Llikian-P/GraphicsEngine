# GraphicsEngine

## What is it about ?
This project is an implementation of a graphics engine using OpenGL with GLAD and GLFW.

## Setup

### Compile & Run
From project root :
```bash
cmake -B build && cmake --build build
```

### Run
```bash
bin/GraphicsEngine
```
